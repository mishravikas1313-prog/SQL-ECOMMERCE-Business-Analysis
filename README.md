# 🛒 SQL Business Analysis on Brazilian E-Commerce Dataset

A complete SQL project built using PostgreSQL to solve real-world business problems on the Brazilian Olist E-Commerce dataset.

This project demonstrates SQL skills ranging from basic querying to advanced analytics using joins, CTEs, subqueries, window functions, and views.

---

## 📌 Project Objectives

- Design relational database schema
- Import and clean raw CSV data
- Perform business analysis using SQL
- Solve real-world analytical questions
- Practice advanced PostgreSQL features

---

## 🛠 Tech Stack

- PostgreSQL
- pgAdmin 4
- SQL

---

## 📂 Dataset

**Dataset:** Brazilian E-Commerce Public Dataset by Olist

It contains information about:

- Customers
- Orders
- Sellers
- Products
- Payments
- Reviews
- Geolocation
